# Cloud File Optimizer & Smart Vault

A Dropbox-style cloud storage app with a twist: automatic file optimization
(compression, deduplication) and a smart, secure vault for sensitive files.

## Tech Stack
- **Frontend:** Node.js / TypeScript
- **Backend:** Python
